StartupPad

A launchpad built specifically for startups moving from Web 2.0 to Web 3.0 using Stellar Blockchain.

Get Started

Built for Startups

StartupPad is created as an opportunity for all startups in the world that in their life cycle have worked from a Web 2.0 perspective, and require moving to a world with more opportunities than Web 3.0 provides.

Web 2.0 Model is broken

The current model of the Web, which is strongly supported by Venture Capital, has already reached its useful life and requires a renewal, not only for the new companies that are born in Web 3.0, but also for the startups already in operation that need move to a decentralized world with a data economy that empowers them.

Launchpad

Our proposal consists of a Launchpad for Startups, which allows a transition of certain models towards decentralized technologies that can be controlled by their users, giving financing to companies and empowering investors. The advantage is that we are a bridge between startups and other launchpads exclusively dedicated to working with 100% blockchain projects.

Startup Token

Our token (STARTUP) allows the governance of the system, deciding which startups can participate in an ITO (Initial Transition Offering), being part of different startups (such as other Venture Capitals) and obtaining liquidity both in the sale and purchase of the token (something that a VC cannot do immediately).
